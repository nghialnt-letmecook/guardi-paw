## Getting Started

To run the project locally:

```bash
npm install
npm run dev
# or
yarn install
yarn dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Main application code
- `components/` - Reusable UI components
- `public/` - Static assets
- `styles/` - CSS and styling
